# Vehicle Service Management System

## Features
- Service Booking
- Live Tracking
- Billing & Payments
- Admin Dashboard
- Mechanic Panel

## Tech Stack
- React (Vite)
- Node.js + Express
- MySQL

## Setup Instructions
1. Import database.sql into MySQL
2. Go to backend → npm install → npm start
3. Go to frontend → npm install → npm run dev
4. Open http://localhost:5173
